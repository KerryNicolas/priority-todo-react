# Priority Todo React App

Une application Todo simple et moderne, construite avec React, Tailwind CSS et DaisyUI.

## Fonctionnalités
- Ajouter des tâches avec priorité (Urgente, Moyenne, Basse)  
- Filtrer les tâches par priorité  
- Sauvegarder les tâches avec localStorage  
- Marquer les tâches comme terminées  
- Supprimer des tâches  
- Afficher un message si la liste est vide  

## Technologies
- React  
- Tailwind CSS  
- DaisyUI  
- Lucide Icons  

## Installation
```bash
npm install
npm run dev
